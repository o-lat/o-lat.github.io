How to use the map:
Directions:
To view directions for a shop within the map, touch on the desired shop then directions will appear on the screen.

Zoom:
You can zoom on the map by using the plus and minus buttons located above the map. Then to move around the map use the scroll bars located on the bottom and right hand side of the map (only appear once the map has been zoomed).

Categories:
To view categories for different shops, use the navbar at the top of the map page and select the category which you want.

Search:
To search for shops in the Shopping Center, go to the directories page (located under the drop down menu on the main menu) and use Ctrl + F on the keyboard then type the shop you want to search for and press Enter on the keyboard (if necessary).
